LED Bloom Bars — band-mapped + random + bass bloom

Each bar's base signal is a fixed band position around the ring (bass on one arc, treble opposite), plus a persistent per-bar random drift so it doesn't read as a mechanical EQ. A bass-onset detector fires a 5-bar bloom impulse (center highest, ±1 lower, ±2 lower still) into the same spring physics every bar already uses — the bounce preset decides how that impulse rises and settles.

Fake synthetic beat (128 BPM kick/snare/hats) drives all 12 tiles identically; only bounce preset and color differ tile-to-tile. The small white flash marks exactly which bar the hit centered on, angle-locked, so the bass-arc mapping is checkable, not just asserted.
